# HR SaaS 評価モジュール - 評価結果一覧画面

HRSaaSプロダクトにおける評価モジュールのプロトタイプです。従業員の評価結果を一覧表示し、効率的に管理できるWebアプリケーションです。

## 🎯 主な機能

### 📊 評価結果一覧表示
- 従業員の評価データを見やすい表形式で表示
- 従業員名、部署、評価期間、評価タイプ、総合スコア、ステータスを一覧表示
- レスポンシブデザインで様々なデバイスに対応

### 🔍 検索・フィルタリング
- **リアルタイム検索**: 従業員名、部署、従業員IDで即座に検索
- **ステータスフィルター**: 完了、レビュー中、未完了、保留で絞り込み
- **評価期間フィルター**: 評価期間ごとにデータを表示
- **部署フィルター**: 特定の部署の評価結果のみを表示
- **ワンクリックリセット**: すべてのフィルターを一括クリア

### 🔢 ソート機能
- 従業員名、部署、評価期間、総合スコアでソート可能
- 昇順・降順の切り替えに対応
- 視覚的なソートインジケーター

### 📈 統計情報
- 総評価数
- 完了した評価数
- レビュー中の評価数
- 平均スコア
- リアルタイムで更新される統計カード

### 📄 ページネーション
- 10件ずつの分割表示
- ページ間のスムーズな移動
- 現在のページと総ページ数を表示

### 🔍 詳細表示
- モーダルウィンドウで評価の詳細を表示
- 基本情報、評価スコアの内訳、ステータス、コメントを表示
- 技術力、コミュニケーション、リーダーシップ、問題解決力、チームワークの5項目を個別表示

## 🚀 使用方法

### 1. ローカルサーバーの起動

このプロトタイプを表示するには、ローカルサーバーが必要です。以下のいずれかの方法で起動できます：

#### Python 3を使用する場合
```bash
cd evaluation-module
python3 -m http.server 8000
```

#### Node.jsのhttp-serverを使用する場合
```bash
cd evaluation-module
npx http-server -p 8000
```

#### VS Codeの Live Server拡張機能を使用する場合
1. VS Codeで `index.html` を開く
2. 右クリックして「Open with Live Server」を選択

### 2. ブラウザでアクセス

ブラウザで以下のURLにアクセス：
```
http://localhost:8000
```

## 📊 モックデータ

このプロトタイプには50件のモック評価データが含まれています：
- 従業員名: ランダムな日本人名
- 部署: 営業部、開発部、マーケティング部、人事部、企画部
- 評価期間: 2023年度下期〜2024年度下期
- 評価タイプ: 自己評価、上司評価、360度評価、プロジェクト評価
- スコア: 70〜100点のランダムスコア
